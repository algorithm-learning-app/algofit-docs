# 게임화 (Gamification)

MVP는 **가볍게**: XP, 스트릭, 하트, 뱃지. 리그·글로벌 랭킹·PvP는 제외.

---

## XP (경험치)

### Daily 챌린지 — 현재 구현 기준

| 이벤트 | XP |
|--------|-----|
| Daily 문항당 (정답 무관, 응답 기준) | +10 |
| Daily 만점(5/5) 보너스 | +20 (`dailyPerfectBonusXp` / `DAILY_PERFECT_BONUS_XP`) |

→ 5문항 완료 시 기본 **50 XP**, 만점이면 **70 XP**. (별도 "세션 완료"·"챌린지 완료" 보너스는 Daily에 미적용)

### 기타 모드 — MVP 가정 (구현 시 조정)

| 이벤트 | XP |
|--------|-----|
| Pick 정답 (Stage/Algorithm) | +10 |
| Blank 정답 (문항 기준) | +15 |
| Blank 빈칸 1개만 맞음 | +5 (문항 미완료 시) |
| 스테이지 최초 클리어 | +25 |

- 레벨(유저 Lv)은 **누적 XP**로 산출 (예: Lv n = 100 × n² XP 필요 — 구현 시 조정)
- MVP: 레벨 UI는 숫자만, 보상 해금 없음

**Later**: 주간 XP 목표, 친구와 비교(비동기)

---

## Streak (연속 학습일)

| 규칙 | MVP |
|------|-----|
| 인정 조건 | 해당 **로컬 날짜**(Asia/Seoul)에 Daily challenge **완료** (5문항 응답) — **정답률 무관** |
| 끊김 | 자정 이후 미완료 시 0으로 리셋 |
| 표시 | 홈 불꽃 아이콘 + 숫자, 완료 화면 “N일 연속” |

### 설계 의도 (사용자 친화성)

- 스트릭은 **꾸준함**의 보상이지 **정답률**의 보상이 아닙니다. 1문제만 맞아도, 0문제를 맞아도 5문항을 끝까지 풀었다면 스트릭 +1.
- 만점(5/5)은 별도 보상 트랙:
  - **`perfect_daily` 뱃지** (해당 일 1회 획득)
  - **보너스 +20 XP** (`dailyPerfectBonusXp`)
- 이로써 `02-core-learning-loop.md`의 "실패 비용 낮음" 원칙과 정렬됨. 잘 못 푼 날에도 스트릭이 끊기지 않아 좌절 줄임.

- **스트릭 실드** (1회 하루 건너뛰기 방어): Later
- 푸시 알림 “오늘 챌린지”: Should (권한 요청)

---

## Hearts (하트)

| 규칙 | MVP |
|------|-----|
| 최대 | 5 |
| 소모 | Level/Algorithm 모드에서 **오답 1회** = -1 |
| 회복 | 30분당 1개 또는 “내일 5개로 충전” (단순: 4시간마다 1개) |
| Daily | 하트 소모 없음 |

- 0개일 때: Level/Algorithm 진입 불가, Pick 연습·Daily는 가능
- **무한 하트 구독**: Later (수익화 후보)

---

## Badges (뱃지)

| 뱃지 ID (예) | 조건 | MVP |
|--------------|------|-----|
| `first_session` | 첫 세션 완료 | ○ |
| `streak_3` | 스트릭 3일 | ○ |
| `streak_7` | 스트릭 7일 | ○ |
| `world1_clear` | World 1 클리어 | ○ |
| `pattern_hash_10` | hash 태그 정답 10 | Should |
| `perfect_daily` | Daily 5/5 전부 정답 (해당 일 1회) — **+20 XP 보너스 동시 지급** | ○ |

- 프로필에 그리드 표시, 미획득은 실루엣
- **Later**: 시즌 뱃지, 희귀도

---

## MVP vs Later 요약

| 요소 | MVP | Later |
|------|-----|-------|
| XP / Lv | ○ | 시즌 패스, 코스메틱 |
| Streak | Daily만 | 실드, 친구 스트릭 |
| Hearts | ○ 단순 회복 | 구독, 광고 회복 |
| Badges | 5~8개 | 많은 수집, 업적 체인 |
| League / Ranking | ✗ | ✗ (명시적 비목표) |
| Leaderboard | ✗ | 주간 비동기 (검토) |

---

## UX 주의

- 하트 0일 때 **비난 없이** “잠시 쉬거나 Daily를 해보세요”
- XP 팝업은 0.8초 이하, 연속 문항 시 누적 표시 가능
- 아동·저연령 미타깃: 과금 압박 UI 없음 (MVP 무과금)
