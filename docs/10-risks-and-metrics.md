# 리스크 · 지표 · 수익 방향

## 북극성 지표 (North Star)

**주간 활성 학습자(WAL) 중 “Daily 완료” 비율**

- WAL: 해당 주에 1회 이상 세션 완료한 사용자  
- 분모·분자 모두 게스트 포함  
- 이유: 습관 형성이 제품 가치와 직결

보조 지표: D1/D7 리텐션, 스트릭 7일 도달률, World 1 클리어율

---

## 리스크

| 리스크 | 영향 | 완화 |
|--------|------|------|
| 콘텐츠 제작 병목 | 출시 지연 | 스키마 고정, 샘플 10문항→템플릿 복제 |
| “암기 앱” 인식 | 이탈, 평판 | Pick 지문을 상황 중심으로, 해설 품질 QA |
| 백준과 중복 논쟁 | 포지셔닝 혼란 | 마케팅: 워밍업·패턴 먼저 ([01-product-vision](01-product-vision.md)) |
| 게스트 데이터 유실 | 이탈 | 로컬 우선 + 서버 merge, 업그레이드 CTA |
| Blank 오답지 다양성 부족 | 너무 쉬움 | distractor 가이드, 사용자 오답 로그 |
| 하트로 인한 좌절 | D1 하락 | Daily 무하트, 0일 때 연습 모드 안내 |
| 저작권 (코드 스니펫) | 법적 | 자체 작성 스켈레톤, CC0 스타일 |

## 수익화 방향 (brief)

MVP: **무과금**. 검증 후 아래 순서 검토.

| 단계 | 모델 | 비고 |
|------|------|------|
| 1 | 구독 (무한 하트, 통계) | 학습 방해 최소화 위해 Pay-to-win 회피 |
| 2 | 일회성 “World 패키지” | 콘텐츠 확장 시 |
| 3 | B2B (부트캠프 라이선스) | 클래스 진행 대시보드 |

하지 않을 것 (원칙): 문제집 랭킹 결제, 리그 입장권.

## 성공 / 실패 기준 (6개월 가정)

| 신호 | 기준 |
|------|------|
| Go | D7 ≥ 15%, Daily 완료 WAL 비율 ≥ 40% |
| Pivot | D1 < 25% → 세션 길이·첫 문항 난이도 |
| Stop | 8주 연속 WAU 정체 + 콘텐츠 소진 |
