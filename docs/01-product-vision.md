# 제품 비전

## 한 줄 정의

**3~5분 세션으로 알고리즘 패턴 인식을 훈련**한다. 처음부터 긴 문제 풀이가 아니라, “이 상황에 어떤 패턴?”을 빠르게 맞히는 능력을 키운다.

## 문제 인식

| 페인 | 설명 |
|------|------|
| 진입 장벽 | 백준·프로그래머스는 “풀이” 중심이라 막히면 이탈 |
| 패턴 부재 | 문제는 많이 풀었는데, 유형 분류·선택이 느림 |
| 시간 부족 | 하루 30분 이상 투자하기 어려움 |
| 동기 부족 | 성취감·습관이 쌓이기 전에 포기 |

## 솔루션

- **짧은 자극**: 한 문항 30~90초 목표
- **선택·빈칸 중심**: 구현 부담을 낮추고 패턴 매칭에 집중
- **즉시 피드백**: 정답 이유를 2~3문장으로
- **가벼운 게임화**: XP, 스트릭, 하트 — 리그·랭킹은 MVP 제외

## 타깃 사용자

| 세그먼트 | 니즈 |
|----------|------|
| 코딩 테스트 입문자 | “어디서부터?” → 패턴 이름부터 익히기 |
| 주니어 | 실무는 되는데 알고리즘 유형 선택이 느림 |
| 알고리즘 거부감 | 긴 풀이 대신 성공 경험을 쌓고 싶음 |

비타깃 (MVP): ACM 올림피아드 준비생, 이미 고난도 DP·세그트리를 다루는 사용자.

## 포지셔닝: 백준 / 프로그래머스 vs 본 앱

| | 백준·프로그래머스 | 본 앱 |
|--|-------------------|--------|
| 단위 | 문제 1개 = 풀이 완료 | 문항 1개 = 패턴 1회 인식 |
| 성공 기준 | AC | 정답 + 짧은 해설 이해 |
| 시간 | 20분~수시간 | 3~5분 세션 |
| 역할 | **실전 풀이 연습장** | **패턴 인식 워밍업** |
| 연결 (Later) | — | “이 패턴으로 풀 문제” 브릿지 |

본 앱은 경쟁·대체가 아니라 **앞단 워밍업**이다. “여기서 패턴 이름을 익히고, 익숙해지면 백준으로”가 장기 내러티브다.

### 차별화: 놀이 → “회사는 이렇게 묻는다”

Pick·Blank는 **알고리즘 언어**로 패턴을 익힌다. **실전 시나리오** 모드는 같은 능력을 **서비스·업무 맥락 지문**으로 옮긴다 — 코딩 테스트에서 가끔 나오는 “현실 문제” 유형을, 기출 암기가 아니라 **매핑 연습**으로 다룬다.

| 단계 | 앱에서의 경험 |
|------|----------------|
| 1 | Pick/Blank로 패턴 이름·골격 (가벼운 게임 톤) |
| 2 | 실전 시나리오로 지문 번역 (“배송 최소 칸” → BFS 신호) |
| 3 (Later) | 외부 플랫폼 실제 풀이 브릿지 |

회사 이름·실제 기출을 복제하지 않고, **일반화된 상황극**으로 재미와 현실감을 동시에 잡는다 ([12-real-world-scenarios](12-real-world-scenarios.md)).

## 핵심 가치

1. **부담 없음** — 틀려도 10초 안에 다시 시도 가능 (하트 규칙 내)
2. **명확함** — 정답 = “이 패턴” + 한 줄 이유
3. **습관** — 일일 챌린지·스트릭으로 매일 조금
4. **재미** — 월드 맵, 진행률, 뱃지 (과하지 않게)

## 성공 지표 (초기)

| 지표 | 정의 | MVP 목표 (가정) |
|------|------|-----------------|
| D1 리텐션 | 가입/첫 실행 후 다음 날 재방문 | ≥ 35% |
| D7 리텐션 | 7일 후 재방문 | ≥ 15% |
| 일일 세션 완료율 | 일일 챌린지 시작 → 5문항 완료 | ≥ 60% |
| 평균 세션 길이 | 앱 foreground 시간 | 4~7분 |
| 스트릭 3일+ 비율 | 연속 3일 일일 완료 사용자 비율 | 측정 후 기준 설정 |

북극성 지표 상세는 [10-risks-and-metrics.md](10-risks-and-metrics.md)를 참고한다.

## 제품 원칙 (의사결정 시)

1. **패턴 먼저, 구현 나중** — Blank도 5~15줄 스켈레톤 수준
2. **한 세션 = 한 승리** — 세션 끝에 XP·완료 체크 반드시
3. **설명은 짧게** — 피드백 본문 3문장 이내 권장
4. **MVP는 좁게** — Python, 6패턴, 2월드

## 비전 (12개월)

“알고리즘 단어장(voca)”처럼, 출근 전 5분이면 이번 주 코딩 테스트 유형을 **이름으로** 말할 수 있는 상태. 이후 실제 문제 플랫폼으로 자연스럽게 이어지는 브릿지.
